Photo import tips
  • Quick import: choose image files already named with student names, such as 6_Last_First_7.jpg or Last_First_7_6.jpg.
  • HTML key import: choose the whole-school HTML key and photo folder, then attach only the matches for the current dashboard roster.
  • Advanced photo tools: Advanced photo tools can create a renamed ZIP copy of the full photo folder. This only exports renamed files; it does not add students to your dashboard roster.
  • Imported images are resized before saving. Use Export Full Dashboard Backup if you want attached photos preserved in a backup file.

Data & Templates Build 16.15.1

Use this space to move, restore, or reuse dashboard data. Data is saved in this browser for this app location; export files only when you intentionally want a copy.
Full Dashboard Backup

Use this when you want an exact copy of the current dashboard, including students, job assignments, seating plans, photos, today's reminders, absences, and daily completion states.

Privacy note: Full dashboard backup files may include student names, seating plans, photos, jobs, reminders, and classroom setup details. Store or share backup files carefully.

Opening the dashboard in a different browser, on another device, or from a different app location may start fresh. Use Backup and Restore here when you intentionally want to move your full setup.

Templates

Use this to save reusable setup work, especially job descriptions and colours. Templates do not include students or daily status unless a future version explicitly asks you to include them.

Templates are useful for sharing structure without sharing a class roster. Template exports are named differently from full backups so they are easier to recognize.

Saved local templates
These are saved in this browser for this app location. Export a template file if you want to archive it or share it with another teacher.
Templates apply reusable structure such as weekly schedule, job names, descriptions, colours, On Call settings, break ranges, and display preferences. Your current student roster stays unchanged by default. Backups are different: they are meant to move or restore a whole dashboard and may include student data.
